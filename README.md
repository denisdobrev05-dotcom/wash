# Миене на DS3 🚿

Инсталируемо **PWA** приложение — интерактивен воден гид за миене на Citroën DS3
стъпка по стъпка. Чист HTML/CSS/JavaScript, без framework. Mobile-first, светла
и тъмна тема, изцяло на български, работи **офлайн**.

## Инсталиране на iPhone

1. Отвори линка към GitHub Pages в **Safari**.
2. Бутон **Сподели** → **Добави към началния екран**.
3. Пусни го от иконата — отваря се на цял екран като приложение и работи офлайн.

## Функции

- Стартов екран с избор **WetCoat ден / само миене** (запомня се в `localStorage`).
- Една стъпка на екран с прогрес лента, части **А/Б/В/Г** с цветово кодиране,
  съвети (💡) и предупреждения (⚠️).
- Обратен таймер **1:30** на стъпката „Изчакай пяната“.
- Финален екран с обобщение и „Петте правила“.

## Файлове

| Файл | Роля |
|------|------|
| `index.html` | Разметка + PWA/iOS meta тагове, регистрира service worker |
| `styles.css` | Стилове, светла/тъмна тема (`prefers-color-scheme`) |
| `app.js` | Логиката на стъпките, таймера и екраните |
| `sw.js` | Service worker — офлайн кеш (cache-first) |
| `manifest.webmanifest` | PWA манифест (относителни пътища за GitHub Pages) |
| `icons/` | Икони 192/512 + apple-touch-icon 180 |
| `generate_icons.py` | Скрипт, генерирал иконите програмно (Pillow) |

Всички пътища са относителни (`start_url: "."`, `scope: "./"`), за да работи
и под подпапка (project page на GitHub Pages).

## Локално стартиране

```bash
python3 -m http.server 8000
# после отвори http://localhost:8000/
```

> Service worker изисква `http(s)` (не `file://`).
